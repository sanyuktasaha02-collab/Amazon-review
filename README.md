PROJECT TITLE
Comparative Analysis of Machine Learning Models for Sentiment Classification of Amazon Alexa Reviews

ABSTRACT
With the rapid growth of e-commerce platforms, customer reviews have become an important source of information for both consumers and manufacturers. Manual analysis of thousands of reviews is time-consuming and inefficient. This project develops a machine learning-based sentiment analysis system to automatically classify Amazon Alexa customer reviews as positive or negative.

INTRODUCTION
Customer reviews significantly influence purchasing decisions. Sentiment Analysis is a Natural Language Processing (NLP) technique that extracts opinions and emotions from textual data.

OBJECTIVES
- Perform exploratory analysis of Amazon Alexa reviews.
- Preprocess textual review data.
- Convert text into numerical features.
- Train machine learning models for sentiment classification.
- Compare model performance.
- Save the trained model for future deployment.

DATASET DESCRIPTION
Dataset: Amazon Alexa Reviews Dataset

Target Variable:
feedback = 1 -> Positive Review
feedback = 0 -> Negative Review

METHODOLOGY
1. Data Cleaning
2. Text Preprocessing
3. Stopword Removal
4. Stemming
5. Count Vectorization
6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Model Saving

MODELS USED
- Random Forest Classifier
- XGBoost Classifier

EVALUATION METRICS
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

CONCLUSION
The project successfully developed a sentiment analysis system for Amazon Alexa reviews using machine learning techniques.
